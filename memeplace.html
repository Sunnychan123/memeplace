Daily Meme
Every user is given the same meme template each day and captions the Meme. To encourage the users to spread innovative and funny ideas among MemePlace
Meme Arena
Users can upload a self-designed meme which will then be put into an 'arena'. Another challenger can personalize the Meme and start a 1vs1 battle by responding to each other by using Meme only. Other participants can give a like to the preferred one, and the player in 'arena' with more impressions wins the battle. Each 'arena' competition lasts long for a week.
Meme Matching
A user can upload a self-designed meme and have a chance to start the matching of a stranger in MemePlace once per day. Once the users like their own memes and want to know more about each other, they can kick off the communications in a private chatroom by accepting the invitations
